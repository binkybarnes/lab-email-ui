How should "rerun the AI" work?

After the first email is generated, how does the user customize or try again?

A

Tone Picker + Regenerate

A small row of toggles before generating: Formal / Casual / Enthusiastic + length Short / Medium / Long. User tweaks these and hits Regenerate to get a new version.

Formal Casual Enthusiastic  ·  Short Medium Long
B

Freeform Instruction Box

A small text field: "Any instructions? (optional)" — e.g. "mention my Python skills", "keep it under 150 words", "focus on their cancer metabolism work". User types what they want and hits Generate.

"make it shorter and mention my cell biology background"
C

Both — Tone Toggles + Instruction Box

Tone/length chips for quick adjustments, plus an optional freeform box for specific requests. More control, slightly more UI.
